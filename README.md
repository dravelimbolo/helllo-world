# SkillHub - Landing Page

Landing page publique de la plateforme interne helllo-world.

## Demo en ligne
https://dravelimbolo.github.io/helllo-world/

## Stack
- HTML5 semantique
- CSS3 (Flexbox, Grid, variables)
- JavaScript Vanilla ES6+ (fetch AJAX, validation formulaire)

## Lancer le projet en local
```
npx serve .
```

## Structure
```
helllo-world/
- index.html
- css/style.css
- js/app.js
- data/formations.json
- assets/
```
